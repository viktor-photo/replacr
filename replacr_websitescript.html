Link replacer website script

If you have any problems, please contact nk-gcash in Skype.

Mode 0 - replace all links on a website

If you are using this mode, please make sure to exclude your own website so the onpage navigation still works.

For example you will run this script on www.myblog.com, then exclude myblog.com.

You can also include subdomains of previously excluded domains.
For example you excluded myblog.com, but want all URLs linking to myblog.com/download to be replaced. Then include myblog.com/download.

Mode 1 - replace only included links

For example you only want uploaded.to links to be replaced, then simply include uploaded.to.

If you are not excluding or including a domain, simply leave it blank.
